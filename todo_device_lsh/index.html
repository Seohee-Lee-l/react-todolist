To-do list (React)

할 일 관리 웹앱

React를 활용해 할 일 추가, 완료, 삭제 기능을 구현한 간단한 일정 관리 프로젝트입니다.

프로젝트 진행자: 이서희

기획 의도

React의 상태 관리, 컴포넌트 구조를 이해하기 위해 제작한 할 일 관리 앱입니다. 로컬 스토리지를 활용해 새로고침 후에도 데이터가 유지되며, 완료 항목 숨기기·전체 삭제·목록 제거 등 실사용에 필요한 기능을 구현했습니다.

작동 원리

할 일 목록을 상태(State)로 관리

  • 데이터 영속성: localStorage에 할 일 목록 데이터를 저장하고 불러옵니다.
  • 상태 변경: 사용자가 항목을 추가, 체크, 삭제할 때 useState를 사용해 데이터를 변경합니다.
  • 렌더링: 상태가 변경될 때마다 useEffect 훅을 통해 변경된 목록을 업데이트하고 localStorage에 저장합니다.
